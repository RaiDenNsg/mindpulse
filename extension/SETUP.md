# Quick Start Guide - MindPulse Tracker Extension

## Step 1: Load the Extension in Chrome

1. Open Chrome browser
2. Go to **`chrome://extensions/`** (paste this in the address bar)
3. Look at the top right corner
4. Toggle on **"Developer mode"**
5. Click **"Load unpacked"** button that appears
6. Navigate to and select the `extension` folder
7. Click "Select Folder"

✅ The extension should now appear in your extensions list!

## Step 2: Pin the Extension

1. Click the Extensions icon (puzzle piece) in Chrome toolbar (top right)
2. Find "MindPulse Tracker"
3. Click the pin icon to pin it to the toolbar
4. Now you'll see 🧠 icon always visible

## Step 3: Start Tracking

1. Go to:
   - **LeetCode** (leetcode.com)
   - **HackerRank** (hackerrank.com)
   - **Programiz** (programiz.com)

2. Start writing code in their editor
3. Click the 🧠 **MindPulse Tracker** icon in your toolbar
4. Watch your metrics update in real-time!

## What You'll See

**In the Popup:**
- ⏱️ **Keystrokes** - Total characters typed
- ⌨️ **Typing Speed** - Words per minute
- 🎯 **Focus Score** - Quality of your typing (%)
- 🧠 **Cognitive Load** - Mental effort (%)
- ⌫ **Backspaces** - How many times you deleted
- **Session Time** - How long you've been coding

**Status Indicator:**
- 🟢 Green dot = Connected to MindPulse
- Last updated timestamp

## Buttons

- **Reset Session** - Clear all metrics and start fresh
- **Sync to Firebase** - (Coming soon) Save data permanently

## Troubleshooting

**Extension not showing metrics?**
1. Make sure you're on LeetCode, HackerRank, or Programiz
2. Click in the code editor area (the tracking starts after you type)
3. Wait a few seconds for the first metrics update
4. Refresh the popup to see latest data

**Metrics are all zeros?**
- You need to **type something first** in the editor
- Metrics calculate every 30 seconds after you start typing

**Extension icon not visible?**
1. Go to `chrome://extensions/`
2. Find MindPulse Tracker
3. Click the 3 dots menu
4. Select "Pin to toolbar"
5. Reload the page (F5)

**Need to reload the extension?**
1. Go to `chrome://extensions/`
2. Find MindPulse Tracker
3. Click the refresh icon
4. Reload the coding platform page
